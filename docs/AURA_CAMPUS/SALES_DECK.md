## Objection-handling cheat sheet

| Objection | Response |
|-----------|----------|
| "We already have an ERP." | "Most cover admin only. Aura is admissions→alumni **plus** accreditation evidence and every portal — one system, not bolt-ons." |
| "Migration will be painful." | "Bulk CSV import for students & staff, an onboarding wizard, and go-live in a day. See the Onboarding Toolkit." |
| "Is the AI timetable real?" | "Yes — Google OR-Tools, in production on Railway. We'll generate one live in the demo." |
| "What about AI features cost?" | "AI summaries/assistant are an **optional add-on**; the core platform's full value needs no AI credit." |
| "Data privacy / security?" | "Per-institution isolation (tested), DPDP consent & erasure, audit logs, cloud-hosted. No data leaves your tenant." |
| "Too expensive." | "One subscription replaces 8 tools and 40+ staff hours/week. Start with a free 30-day trial before you pay anything." |
